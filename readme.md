# OpenPlatform: Calculator

- supports bundle called `app.bundle`

## Requirements

- Node.js +v10
- Total.js framework +v3.3

## Installation

- __IMPORTANT__: update `openplatform.json` file

```bash
$ cd op_calculator
$ npm install
```
